# Bookbinding, and the Care of Books by Douglas Cockerell (1901)

A version of the Douglas Cockerell's Bookbinding, and the Care of Books,
available as an e-book from
[Project Gutenberg](https://www.gutenberg.org/ebooks/26672/).

This version has been typeset with Typst for printing at home, for DIY
bookbinding.

A scan of the original book is also available at the
[Internet Archive](https://archive.org/details/bookbindingcareo00cockrich).
